DataBasE

Key-value Database

  • key : value 형태
  • 1차 ram 저장
  • 데이터 캐싱, Pub/Sub, 영상스트리밍, 로그인 기록저장 등
  • 서브 DB

Relational Database

  • 관계형 데이터 베이스
  • 데이터 정규화됨
  • ACID Transaction 기능
  • 엑셀

Graph Database

  • 관계형 DB보다 더 관계형인 관계형DB
  • 비행기 노선, 친구관계, 추천서비스 등
    • *Sparsity

Document Database

  • collection > document > json { key:value }

Column-family Database

  • cql 같은 전용 언어 사용하는 것도 있음
  • 정규화 안함, 분산 복제 잘됨
  • 시계열 데이터 분석
  • 데이터 일관성 부족

Search engine

  • 검색용 인덱스보관용 DB
  • index 보관 특화
  • 실시간 검색어, 추천 검색어, 검색어 오타교정 등
